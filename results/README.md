# Result-source status

`edl-ablation-s18-final-summary.md` and `edl-ablation-s18-final-summary.json` are the canonical indexes for Supplementary Table S18 in release v0.2.8. They reuse the Full EDL rows from the main evidence-seeking experiment and point to the updated SciFact feature-removal traces run with the S14 SciFact configuration.

The older `edl-ablation-scifact-nli4ct-component-summary.*` files and the shorter-named SciFact ablation directories are retained only as historical provenance from earlier exploratory runs. They are not sources for the v0.2.8 manuscript tables or conclusions.
